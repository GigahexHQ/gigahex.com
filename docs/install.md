### Pre-requisite

:::caution

Gigahex is curently in alpha, available to experiment with Docker.

:::

Docker must be installed in your desktop or VM. Run the following command to get
up and running with Gigahex.

```bash
docker run --rm -it -p 9080:9080 \
  --name gigahex \
  -v gxdb:/var \
  -v gxbin:/usr/local \
  gigahex/gxc:0.1.0
```
